# HƯỚNG DẪN CHẠY & BUILD WEB-CLIENT

## Windows

### 1) Cài package còn thiếu
```bash
cd web-client
npm install
```

### 2) Chạy dự án (dev)
```bash
cd web-client
npm run dev
```

### 3) Build dự án
```bash
cd web-client
npm run build
```
